Inserindo Vídeos Hospedados Localmente

Este vídeo está hospedado no meu servidor.

Para adicionar vídeos, usar a tag VIDEO e os parâmetros: width= para definir a largura; poster= para definir uma imagem inicial; controls e autoplay para a execução do vídeo.

Depois, entre as tags VIDEO adicionar uma tag source:src onde ficarão os vídeos nos diferentes formatos. A ordem é importante.
