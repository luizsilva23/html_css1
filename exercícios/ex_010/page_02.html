É importante usar o parâmetro rel='next' ou rel='prev' para que os mecanismos de buscas entendam qual é a página seguinte do site e qual é a anterior.

Esta página é um oferecimento GUILHERMELEFAY

O parâmetro rel='nofollow' é usado para quando apesar de colocar um link para a página, não queremos nos associar a ela e dar a entender aos mecanismos de busca que somos associados.

O parâmetro target='_self' é usado para indicadar qua a página abrirá na mesma aba.
