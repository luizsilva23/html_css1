Parágrafos e Linhas

Testando um teste com parágrafos. Este é um exercício do curso em vídeo sobre html.

Para quebras de linhas,
usar sempre a tag <br>".

Vamos adicionar alguns símbolos:
¥ δ Δ ↑ ↕ © ∅

Para usar símbolos, usar o &.

Para adicionar emojis, pesquisar o código e usar com '&#x'.
Ex:. &#x 1 F 3 E 1 = 🏡
&#x 1 F 3 8 4 = 🎄
