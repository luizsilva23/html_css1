Imagens Dinâmicas

Tente abrir esse site em dispositivos diferentes ou apenas aumente e diminua o tamanho do seu navegador.

A tag PICTURE funciona para adicionar imagens dinâmicas. Dentro dela podemos colocar a imagens e os parâmetros de source media type para definir mais tamanhos da imagem, diminuindo ou aumentando de acordo com o tamnho da tela do usuário. Ajustar o parâmetro max width para definir os tamanhos máximos onde cada mudança de imagem deve ocorrer. Prestar atenção na ordem, os menores parâmetros devem vir primeiro no código.

Reproduzindo áudio

Para inserir um áudio, usar a tag AUDIO e configurar os parâmetros controls, para mostrar o botão de play, e o autoplay. Em alguns navegadores, esses parâmentros podem não funionar.

Para adicionar outras opções de formato de áudio para o navegador tentar carregar, utilizar a tag AUDIO e inserir o parâmetro source:src, assim, podemos dar outras opções caso o navegador não seja compatível com a primeira. A ordem aqui também é importante, assim como os parâmetros de configuração: preload, para carregar previamente; controls, para mostrar o botão play e altoplay. em alguns navegadores, eles podem não funcionar. O parâmetro

Os formatos compatíveis com o HTML são o MP3, WAVE e OGG. Evitar os WAVE, pois são muito pesados.
